# Geospatial Temporal Analysis

Análisis geoespacial basado en series temporales satelitales para el monitoreo de variables ambientales y la detección de patrones espaciales.

## Metodología

- Series temporales de imágenes satelitales
- Cálculo de índices espectrales
- Análisis de variabilidad espacial
- Detección de anomalías ambientales
- Integración de variables atmosféricas

## Casos de análisis

Entre los análisis desarrollados se incluyen:

- Monitoreo de calidad del aire
- Detección y seguimiento de incendios
- Identificación de cuerpos y masas de agua
- Evaluación de cambios ambientales en el tiempo
- Análisis territorial en escenarios críticos

## Herramientas utilizadas

- Google Earth Engine
- Python
